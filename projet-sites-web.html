Création de sites web responsives

Contexte du projet

Ces projets se sont déroulés sur les semestres 1 et 2 du BUT Informatique. L'objectif était d'apprendre à créer des sites web modernes, responsives et accessibles en utilisant différentes technologies.

Projets réalisés

  • Projet 1 (S1) : Site vitrine avec WordPress pour une association locale
  • Projet 2 (S1) : Portfolio personnel en HTML/CSS pur
  • Projet 3 (S2) : Site e-commerce responsive avec JavaScript

Compétences acquises

HTML5 & CSS3

  • Structure sémantique avec balises HTML5
  • Layouts avec Flexbox et CSS Grid
  • Animations et transitions CSS
  • Media queries pour le responsive

JavaScript

  • Manipulation du DOM
  • Gestion des événements
  • Fetch API pour les requêtes
  • Validation de formulaires
